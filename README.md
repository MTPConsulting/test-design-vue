# Test design vue

### Requerimientos

1. Node.js

### Instalar vue-cli y crear el proyecto

Ejecutar:

    npm install -g @vue/cli
  
Crear el proyecto:

    vue create project_name

### Descripción del test
Realizar una página similar a la que se muestra en la imagen adjunta. La idea es valorar las habilidades en el uso de los estilos y maquetación de los elementos y no la funcionalidad del mismo.
La parte superior contiene un pequeño formulario, con varios inputs, y dos botones. (No es necesario que funcione, pero sí que el código sea el de un formulario y que sea responsive).
La parte de abajo de contiene una tabla con color en el head y footer, iconos, y una primera columna con un elemento de color (podría ser un chip  o label). Este elemento y la fila en la que se encuentra tiene que tener un color que se pueda cambiar por parámetro.
